About this project:

This is the cl-string-match project ("clstringmatch")

Common Lisp provides a SEARCH function to lookup given keys in a string, but its implementation can be subotimal: it can be beaten in some cases by even the simpliest brute-force algorithm. This project is aimed at implementing robust substring search algorithms for Common Lisp.

Implemented algorithms:

  • Brute-force (also known as na√Øve algorithm)
  • Boyer-Moore (with mismatched character heuristic)
  • Rabin-Karp algorithm
